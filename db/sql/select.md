```aiignore
프로그래머스 : 12세 이하인 여자 환자 목록 출력하기

SELECT PT_NAME
, PT_NO
, GEND_CD
, AGE
, COALESCE(TLNO, 'NONE') AS TLNO
FROM PATIENT
WHERE GEND_CD = 'W'
AND AGE < 13
ORDER BY AGE DESC, PT_NAME ASC
```

```aiignore
SELECT USER_ID
, PRODUCT_ID
FROM ONLINE_SALE
GROUP BY USER_ID, PRODUCT_ID
HAVING COUNT(1) > 1
ORDER BY USER_ID ASC, PRODUCT_ID DESC
```

```aiignore
SELECT ID
, LENGTH
FROM FISH_INFO
WHERE LENGTH > 10
ORDER BY LENGTH DESC, ID
LIMIT 10
```

```aiignore
프로그래머스 : 특정 물고기를 잡은 총 수 구하기

SELECT COUNT(1) AS "FISH_COUNT"
FROM FISH_INFO A
INNER JOIN FISH_NAME_INFO B ON A.FISH_TYPE = B.FISH_TYPE
WHERE B.FISH_NAME IN ("BASS", "SNAPPER") 
```

```aiignore
프로그래머스 : 대장균들의 자식의 수 구하기

SELECT A.ID
, COUNT(B.ID) AS "CHILD_COUNT"
FROM ECOLI_DATA A
LEFT OUTER JOIN ECOLI_DATA B ON A.ID = B.PARENT_ID
GROUP BY A.ID
ORDER BY ID
```
